## WorkSetup
1. Prerequisite Installations
    1. [vs code](https://code.visualstudio.com/Download)
    2. GIT Linux Package 
        $ sudo apt install git-all

3. vs code extensions
    1. Git History
    2. Git Pull and Push Request
    3. pdf Viewer
    4. Markdown enhanced viewer
